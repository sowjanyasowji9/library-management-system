# Library Management System Analysis using SQL

## Project Overview

This project analyzes a Library Management System database using MySQL. The database contains information about books, members, employees, branches, issued books, and returned books. The project focuses on library operations, book tracking, member activity, and business insights using SQL.

## Objectives

- Manage library records
- Analyze book issuance and returns
- Track member activity
- Monitor overdue books and fines
- Generate business insights using SQL

## Database Tables

- Books
- Members
- Employees
- Branch
- Issued Status
- Return Status

## SQL Concepts Used

- SELECT Statements
- INSERT, UPDATE, DELETE
- INNER JOIN
- LEFT JOIN
- GROUP BY
- HAVING
- Aggregate Functions
- CTAS (Create Table As Select)
- Stored Procedures
- DATEDIFF()
- Subqueries

## Key Tasks Performed

- Added new book records
- Updated member information
- Deleted issued book records
- Retrieved books issued by employees
- Created summary tables
- Calculated rental income by category
- Found books not yet returned
- Identified overdue books and fines
- Generated branch performance reports
- Created active member reports
- Automated book issue and return processes using stored procedures
